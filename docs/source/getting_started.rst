================
Getting started
================

This section covers everything you need before writing your first backtest:
how to install QF-Lib, which optional data providers require extra packages,
and how to configure ``settings.json`` and ``secret_settings.json``.

If you are new to the library, read **Installation** first, then **Configuration**.
After that, continue to :doc:`tutorials` for step-by-step guides.
